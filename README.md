CREATE AN FOLDER AS 
" MY ASSISTANT " 
1) mkdir MyAssistant
2) cd MyAssistant  ---> python -m venv venv

   This creates a new folder named venv that will hold all your libraries

   Activate Virtual Environment:
1) .\venv\Scripts\activate

   Install Dependencies
1) pip install speechrecognition
2) pip install pyttsx3
3) pip install pyautogui

   AFTER THIS CHECK ASSISTANT.PY
ASSISTANT WILL WORK WITH NORMAL TASK

YOU CAN DEFINITELY ADD MORE AS YOU NEED :)
